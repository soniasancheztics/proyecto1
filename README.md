# Agència de Viatges

Aplicació web desenvolupada amb PHP que mostra informació sobre diferents destinacions turístiques i permet contactar amb l'agència mitjançant un formulari.

## Descripció del projecte

Aquest projecte utilitza la tècnica d'inclusió de fitxers mitjançant `include` per reutilitzar elements comuns de la interfície, com la capçalera, el menú de navegació i el peu de pàgina.

## Funcionalitats

- Visualització de destinacions turístiques.
- Formulari de contacte.
- Navegació entre pàgines.
- Reutilització de components amb PHP.

## Tecnologies utilitzades

- HTML5
- CSS3
- JavaScript
- PHP

## Instal·lació

1. Copiar el projecte dins del directori `htdocs` de XAMPP.
2. Iniciar Apache.
3. Accedir a:

```text
http://localhost/solucion_2
```

## Objectius didàctics

- Practicar l'ús de PHP.
- Utilitzar fitxers `include`.
- Organitzar un projecte web.
- Crear formularis HTML.
